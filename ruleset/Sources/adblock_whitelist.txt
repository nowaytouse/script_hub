# AdBlock Whitelist - 白名单配置
# 合并规则时自动排除这些域名/规则
# 
# 用途: 防止误拦截导致功能异常
# 例如: 微信内置浏览器的链接重定向被拦截
#
# 格式说明:
#   - 每行一条规则
#   - 支持 DOMAIN, DOMAIN-SUFFIX, DOMAIN-KEYWORD 格式
#   - 以 # 开头的行为注释
#   - 空行会被忽略
#
# 更新: 2025-12-20
# ═══════════════════════════════════════════════════════════════
